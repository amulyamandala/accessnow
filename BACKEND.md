# AccessNow Backend 🖥️

The backend server acts as a proxy between the Chrome Extension and the Google Gemini AI API, handling text processing and summarization requests.

## 🚀 Getting Started

### Prerequisites
-   Node.js (v18+)
-   NPM
-   Google Gemini API Key

### Installation
1.  Navigate to `accessnow-backend/`.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Configure environment:
    Create a `.env` file:
    ```env
    PORT=3000
    GEMINI_API_KEY=your_gemini_api_key_here
    ```
4.  Run the server:
    ```bash
    npm start
    ```

## 🛣️ API Endpoints

### `GET /`
Basic health check to verify server status.

### `POST /summarize`
Generates an AI summary of the provided text.

-   **Body**:
    ```json
    {
      "text": "Full webpage content here..."
    }
    ```
-   **Response**:
    ```json
    {
      "summary": "5 bullet point summary generated by AI"
    }
    ```
-   **Logic**:
    -   Trims input text to 8,000 characters to stay within model limits.
    -   Uses `gemini-2.5-flash` for high-speed, cost-effective summarization.
    -   Instructs the AI to use "simple English" suitable for students.

## 📦 Dependencies

-   **`express`**: Web framework for routing.
-   **`cors`**: Enables requests from the Chrome Extension.
-   **`dotenv`**: Manages environment variables securely.
-   **`node-fetch`**: Handles outgoing HTTP requests to the Gemini API.

## 🧪 Error Handling

-   Returns `400` if no text is provided.
-   Returns `500` if the Gemini API call fails or if there is an internal server error.
