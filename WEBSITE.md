# AccessNow Landing Page 🌐

The AccessNow website is a high-performance, single-page landing site designed to showcase the features and benefits of the AccessNow Chrome Extension.

## 🎨 Design Philosophy

The website follows a modern, dark-themed aesthetic with a focus on:
-   **Visual Clarity**: High-contrast text and a clean grid layout.
-   **Interactivity**: Animated "mock" interfaces that demonstrate the extension's functionality.
-   **Direct CTA**: Clear installation instructions and demo walkthroughs.

## 🏗 Structure

-   **Hero Section**: Introduces the extension's core value proposition: "Summarize, read, and visually adapt."
-   **Features Grid**: Highlights the three main pillars:
    1.  Analyze & Simplify.
    2.  Read Aloud (TTS).
    3.  Visual Accessibility & Dyslexia Support (WCAG 1.4.x compliant).
-   **Accessibility Profiles**: Detailed mapping of features to WCAG 2.1 principles (Perceivable, Operable, Understandable, Robust).
-   **Demo Guide**: Step-by-step instructions for loading the unpacked extension in Chrome.

## 🛠 Tech Stack

-   **HTML5**: Semantic structure for accessibility.
-   **CSS3**: Custom properties (variables), Flexbox, and Grid for a responsive and modern layout.
-   **Responsive Design**: Media queries ensure the site is usable on both desktop and mobile devices.

## 📦 Usage

The page is entirely self-contained in `index.html`. To view it, simply open the file in any modern web browser.
